# TodoMatrix - 产品需求文档 (PRD)
**版本：** v1.0
**目标用户：** 开发者本人
**平台：** macOS 15+ (Sequoia)
**开发语言：** Swift (SwiftUI)
**设计风格：** 原生、现代、纯色背景+彩色边框、支持暗黑模式

## 1. 产品概述
TodoMatrix 是一款为个人量身定制的四象限任务管理工具。遵循“重要/紧急”二维度，采用经典的艾森豪威尔矩阵布局，帮助快速梳理任务优先级。聚焦极简操作和原生体验，所有数据本地存储，无云端同步。

## 3. 功能详细描述

### 3.1 任务管理核心功能

#### 3.1.1 创建任务
- **触发方式**：单击任一象限内的**空白区域**（非任务卡片区域）
- **交互流程**：
  1. 单击空白处，该象限顶部立即出现一个输入框
  2. 输入框自动获得焦点
  3. 输入任务标题，按回车键确认
  4. 任务即创建在当前象限，显示在象限顶部（按创建时间倒序）
  5. 按 ESC 键取消创建
- **快速创建说明**：初始版本只创建标题，其他属性（截止日期、备注、子任务）可通过编辑功能补充

#### 3.1.2 编辑任务
- **触发方式**：双击任务卡片
- **编辑弹窗包含**：
  - 标题（必填，可修改）
  - 截止日期（可选，日期选择器，显示日历）
  - 备注（可选，多行文本区域，支持换行）
  - 子任务列表（可选）：
    - 可添加多个子任务项
    - 每个子任务有独立的复选框
    - 子任务可勾选完成
    - 子任务可删除
- **操作按钮**：保存 / 取消

#### 3.1.3 移动任务
- **方式**：鼠标拖拽
- **交互细节**：
  - 按住任务卡片任意位置开始拖拽
  - 拖拽时卡片半透明，跟随鼠标
  - 拖拽到其他象限区域时，目标象限高亮（边框闪烁或背景微亮）
  - 释放鼠标，任务移动到目标象限
  - 如果拖拽到非象限区域，自动回弹到原位置

#### 3.1.4 完成任务
- **操作**：点击任务卡片左侧的复选框
- **视觉反馈**：
  - 复选框变为选中状态（打勾）
  - 任务卡片执行淡出动画（透明度从1到0，持续0.2秒）
  - 动画结束后，任务从当前象限移除
- **数据变化**：
  - `isCompleted = true`
  - `completedAt = Date()`
  - 任务移入“已完成”列表

#### 3.1.5 删除任务
- **方式**：在编辑弹窗中提供“删除”按钮
- **二次确认**：弹出确认框（“确定删除该任务吗？”）
- **快捷方式**：未来可考虑在任务卡片上右键删除

### 3.2 已完成任务管理

#### 3.2.1 入口
- 窗口左下角固定按钮：“显示已完成”(Show Completed)

#### 3.2.2 已完成视图
- **触发**：点击按钮后，从窗口底部向上滑出一个面板（或弹出独立小窗口）
- **布局**：
  - 按完成时间倒序排列（最新的在上）
  - 每条任务显示：标题、原所属象限（用对应颜色的小圆点标识）、完成时间
  - 任务不可编辑，不可拖拽
- **操作**：
  - 清空全部：右上角“清空”按钮，永久删除所有已完成任务（需二次确认）
  - 单条删除：每条任务右滑或右键删除

#### 3.2.3 关闭已完成视图
- 再次点击“显示已完成”按钮
- 或点击面板外的区域
- 或按 ESC 键

### 3.3 数据持久化

#### 3.3.1 存储方式
- 本地文件存储
- 格式：JSON
- 路径：`~/Library/Application Support/TodoMatrix/tasks.json`

#### 3.3.2 自动保存机制
- 任何操作（增、删、改、移、完成）后立即自动保存
- 应用进入后台时强制保存
- 启动时自动加载

#### 3.3.3 数据迁移
- 版本升级时自动处理数据结构变化
- 无云端同步，无需考虑多设备冲突

## 4. 数据模型

```swift
// 任务模型
struct Task: Identifiable, Codable {
    let id: UUID
    var title: String
    var quadrant: Quadrant
    var dueDate: Date?
    var notes: String?
    var subtasks: [Subtask]?
    var isCompleted: Bool
    let createdAt: Date
    var completedAt: Date?
    
    init(title: String, quadrant: Quadrant) {
        self.id = UUID()
        self.title = title
        self.quadrant = quadrant
        self.dueDate = nil
        self.notes = nil
        self.subtasks = nil
        self.isCompleted = false
        self.createdAt = Date()
        self.completedAt = nil
    }
}

// 子任务模型
struct Subtask: Identifiable, Codable {
    let id: UUID
    var title: String
    var isCompleted: Bool
    
    init(title: String) {
        self.id = UUID()
        self.title = title
        self.isCompleted = false
    }
}

// 象限枚举
enum Quadrant: String, CaseIterable, Codable {
    case q1 = "重要且紧急"
    case q2 = "不重要但紧急"
    case q3 = "重要不紧急"
    case q4 = "不重要不紧急"
    
    var borderPosition: BorderPosition {
        switch self {
        case .q1: return .leading
        case .q2: return .top
        case .q3: return .leading
        case .q4: return .bottom
        }
    }
    
    var color: Color {
        switch self {
        case .q1: return Color.red
        case .q2: return Color.orange
        case .q3: return Color.blue
        case .q4: return Color.gray
        }
    }
}

enum BorderPosition {
    case leading, top, bottom, trailing
}
```

## 6. 交互流程详解

### 6.1 启动
1. 应用启动
2. 加载本地 tasks.json
3. 按象限分组渲染任务卡片
4. 已完成任务不显示在主界面

### 6.2 新建任务
```
用户单击象限空白处
    ↓
输入框出现并获得焦点
    ↓
用户输入标题
    ↓
按回车
    ↓
创建 Task 对象（quadrant=当前象限）
    ↓
添加到当前象限的 tasks 数组
    ↓
UI 更新显示新卡片
    ↓
自动保存
```

### 6.3 完成任务
```
用户点击复选框
    ↓
isCompleted = true
completedAt = Date()
    ↓
启动淡出动画
    ↓
动画结束
    ↓
从当前象限数组中移除
    ↓
添加到已完成数组
    ↓
自动保存
```

### 6.4 拖拽移动
```
用户按住任务卡片
    ↓
开始拖拽，卡片半透明
    ↓
鼠标移入其他象限
    ↓
目标象限高亮
    ↓
释放鼠标
    ↓
更新 task.quadrant
    ↓
从原象限数组移除
    ↓
添加到新象限数组
    ↓
UI 更新
    ↓
自动保存
```

## 8. 非功能性需求

### 8.1 性能要求
- 启动时间 < 1秒
- 任务渲染：1000个任务无卡顿
- 拖拽帧率：60fps
- 内存占用：< 200MB

### 8.2 兼容性
- 仅支持 macOS 15.0+
- 支持 Intel 和 Apple Silicon
- 完美适配暗黑/亮色模式切换

### 8.3 数据安全
- 所有数据仅存本地
- 无网络请求
- 无数据追踪
